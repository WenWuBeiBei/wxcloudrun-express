# 变更日志

所有重要的项目变更都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2026-01-04

### 新增
- 初始化项目框架
- 创建模块化目录结构
- 添加健康检查接口 `/api/health`
- 添加微信 OpenID 获取接口 `/api/wx_openid`
- 创建项目基础文档（README、开发规范、升级指南等）
- 配置数据库连接和 ORM（Sequelize）
- 实现错误处理和 404 处理中间件

### 变更
- 数据库名称从 `nodejs_demo` 更改为 `wenwubeibei`
- 项目名称从 `wxcloudrun-express` 更改为 `wenwubeibei-toolset`
- 重构首页为文武贝贝工具集欢迎页面

### 移除
- 移除计数器 Demo 功能
- 移除 Counter 数据模型

### 技术栈
- Node.js >= 12.0.0
- Express 4.x
- MySQL + Sequelize ORM
- 微信云托管
