# 变更日志

遵循语义化版本。记录对框架和对外接口的关键变更。

## [0.1.0] - 2026-01-04

### 新增
- 提供微信云托管下的基础后端框架，服务名「文武贝贝工具集」
- 内置健康检查 `GET /api/v1/health` 与 OpenID 获取 `GET /api/v1/wx/openid`
- 建立模块化路由与工具模块占位 `GET /api/v1/tools`
- 完整重写欢迎页，突出框架就绪与接入指引
- 编写核心文档：README、API、开发规范、升级指南、快速开始、项目概览

### 变更
- 移除所有 Demo 业务代码，保留最小可运行骨架
- 数据库初始化允许无配置启动，便于本地快速体验

### 移除
- 旧版示例路由与计数器相关描述
