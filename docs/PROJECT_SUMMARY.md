# 项目概览（文武贝贝工具集）

面向微信小程序的多工具后端框架，基于微信云托管 + Node.js/Express。当前版本为最小可运行骨架，专注稳定的基础设施，方便后续按需扩展。

## 目标与定位
- 为「文武贝贝」小程序提供统一的工具集服务端
- 快速扩展：按模块接入多个小工具，复用统一规范
- 云原生：默认适配微信云托管，支持本地快速启动

## 已交付内容（0.1.0）
- 移除所有 Demo 代码，保留最小可运行框架
- 内置健康检查与 OpenID 获取接口，新增工具模块占位路由
- 欢迎页与文档体系重写（README、API、开发规范、升级指南、快速开始）
- 数据库初始化支持无配置启动，便于本地体验

## 目录速览
- `routes/`：API 入口与模块路由（tools 占位）
- `middleware/`：参数校验等通用中间件
- `utils/`：统一响应等工具方法
- `public/`：欢迎页，展示状态与接入步骤
- `docs/`：API、开发规范、升级说明、变更记录

## 下一步路线

2. 设计鉴权与会话方案（结合小程序登录态）
3. 建立日志、监控与告警基线
4. 编写自动化测试与 CI/CD 流程

## 适配微信小程序的关键点
- 仅接受 HTTPS，域名需在「服务器域名」白名单
- 云托管请求头 `x-wx-openid`、`x-wx-source` 可用于用户识别
- 返回体统一 `code + message + data`，便于小程序端处理
